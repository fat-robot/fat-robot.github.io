finanzen.net - Currency Converter

What is finanzen.net Currency Converter?

finanzen.net is Germany's leading financial information portal.

The currency converter is a free tool from Finanzen.net. The app supports over 150 currencies for either the current date or past dates. Conversion rates are stored offline for use when no network is available, and a handy conversion table enables quick lookup of common amounts.

The Project

We were asked by the Finanzen.net team to create an Android version of their currency converter app for iOS. As part of the overhaul of Finanzen's Android offering, the brief was to design and build a Material Design experience to coincide with the launch of Android Lollipop.

Needless to say, the new app would need to flow across multiple screen sizes and support both phones and tablets.

Build

The majority of the build was done using the Android Lollipop preview release. Given the preview was provided many months before the actual launch, there was always a risk that problems would arise once running on the release version of the operating system.
